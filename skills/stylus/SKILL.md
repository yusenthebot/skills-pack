# stylus

## Overview
Robust, expressive, and feature-rich CSS superset. Supporting both an indented syntax and regular CSS style.

## Installation
```bash
npm install stylus
```

## Core API / Usage
```js
$ npm install stylus -g
```

```js
stylus -w style.styl -o style.css
```
