# body-parser

## Overview
Node.js body parsing middleware. Parse incoming request bodies in a middleware before your handlers, available under the `req.body` property.

## Installation
```bash
npm install body-parser
```

## Core API / Usage
```js
$ npm install body-parser
```

```js
const bodyParser = require('body-parser')
```
