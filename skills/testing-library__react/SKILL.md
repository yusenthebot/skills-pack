# @testing-library/react

## Overview
Simple and complete React DOM testing utilities that encourage good testing practices.. As a part of this goal, you want your tests to avoid including implementation details of your components and rather focus on making your tests give you the confidence for which they are intended.

## Installation
```bash
npm install @testing-library/react
```

## Core API / Usage
```bash
npm install --save-dev @testing-library/react @testing-library/dom
```

```bash
yarn add --dev @testing-library/react @testing-library/dom
```

## Common Patterns
### Pattern 1
