# recharts

## Overview
React charts. # Recharts ## Introduction Recharts is a Redefined chart library built with React and D3.

## Installation
```bash
npm install recharts
```

## Core API / Usage
```js
<LineChart width={400} height={400} data={data}>
  <XAxis dataKey="name" />
  <Tooltip />
  <CartesianGrid stroke="#f5f5f5" />
  <Line type="monotone" dataKey="uv" stroke="#ff7300" />
  <Line type="monotone" dataKey="pv" stroke="#387908" />
</LineChart>
```
