# vitest

## When to Use

Invoke when you need to:
- Work with vite
- Work with vitest
- Work with test
- Work with jest

## Quick Start

```bash
npx vitest --help
```

```javascript
import { b, c, g } from 'vitest';
```

## API Reference

**Exported functions:**
- `b`
- `c`
- `g`
- `i`
- `s`
- `v`
- `d`
- `V`
- `a`
- `expectTypeOf`
