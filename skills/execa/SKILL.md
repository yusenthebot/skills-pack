# execa

## When to Use

Invoke when you need to:
- Work with exec
- Work with child
- Work with process
- Work with subprocess

## Quick Start

```javascript
import { execa, execaSync, execaCommand } from 'execa';
```

## API Reference

**Exported functions:**
- `execa`
- `execaSync`
- `execaCommand`
- `execaCommandSync`
- `execaNode`
- `parseCommandString`
- `ExecaError`
- `ExecaSyncError`
- `sendMessage`
- `getOneMessage`
