# tsup

## Installation
```bash
npm install -D tsup
```

## Core Usage

```bash
tsup src/index.ts                        # basic bundle
tsup src/index.ts --format cjs,esm      # dual output
tsup src/index.ts --dts                  # with .d.ts types
tsup src/index.ts --minify              # minified
tsup --watch                            # watch mode
```

## tsup.config.ts

```ts
import { defineConfig } from 'tsup';

export default defineConfig({
  entry: ['src/index.ts'],
  format: ['cjs', 'esm'],
  dts: true,                  // generate .d.ts
  sourcemap: true,
  clean: true,                // clean dist/ before build
  splitting: false,           // code splitting for esm
  treeshake: true,
  minify: false,
  external: ['react'],        // don't bundle peer deps
  noExternal: ['lodash'],     // force bundle this
  target: 'node18',
  outDir: 'dist',
  banner: {
    js: '// Generated by tsup',
  },
});
```

## package.json Setup

```json
{
  "main": "./dist/index.cjs",
  "module": "./dist/index.js",
  "types": "./dist/index.d.ts",
  "exports": {
    ".": {
      "import": "./dist/index.js",
      "require": "./dist/index.cjs",
      "types": "./dist/index.d.ts"
    }
  },
  "scripts": {
    "build": "tsup",
    "dev": "tsup --watch"
  },
  "files": ["dist"]
}
```

## Common Patterns

### Multiple entry points
```ts
export default defineConfig({
  entry: {
    index: 'src/index.ts',
    cli: 'src/cli.ts',
  },
  format: ['cjs', 'esm'],
  dts: true,
});
```

### React component library
```ts
export default defineConfig({
  entry: ['src/index.tsx'],
  format: ['cjs', 'esm'],
  dts: true,
  external: ['react', 'react-dom'],
  jsx: 'react-jsx',
});
```

## Tips & Gotchas
- `dts: true` requires TypeScript — run `tsc --noEmit` separately for type checking
- For dual output, `format: ['cjs', 'esm']` generates `.js` (ESM) and `.cjs` (CJS)
- `splitting: true` only works with ESM format
- `tsup` uses esbuild internally — same performance characteristics
- For libraries, list all dependencies as `external` or `peerDependencies`
