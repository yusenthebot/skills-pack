# axios

## When to Use

Invoke when you need to:
- Work with xhr
- Work with http
- Work with ajax
- Work with promise

## Quick Start

```javascript
import { axios, Axios, AxiosError } from 'axios';
```

## API Reference

**Exported functions:**
- `axios`
- `Axios`
- `AxiosError`
- `CanceledError`
- `isCancel`
- `CancelToken`
- `VERSION`
- `all`
- `Cancel`
- `isAxiosError`
