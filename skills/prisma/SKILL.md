# prisma

## When to Use

Invoke when you need to:
- Work with CLI
- Work with ORM
- Work with Prisma
- Work with Prisma CLI

## Quick Start

```bash
npx prisma --help
```

```javascript
import { prismaVersion, DbNull, JsonNull } from 'prisma';
```

## CLI Reference

```
npx prisma [options]
```

**Options:**
- `--g`
- `--o`
- `--c`
- `--i`
- `--e`
- `--a`
- `--this`
- `--silent`
- `--wait-apps`
- `--background`

## API Reference

**Exported functions:**
- `prismaVersion`
- `DbNull`
- `JsonNull`
- `AnyNull`
- `type`
- `PrismaClient`
- `getLogLevel`
- `dmmf`
- `getPrismaClientClass`
- `NullTypes`
