# terminal-link

## Overview
Create clickable links in the terminal. #### text Type: `string` Text to linkify.

## Installation
```bash
npm install terminal-link
```

## Core API / Usage
```bash
npm install terminal-link
```

```js
import terminalLink from 'terminal-link';

const link = terminalLink('My Website', 'https://sindresorhus.com');
console.log(link);
```
